General information


What is GreyCloud?


GreyCloud is a management software for web services and databases. It is the ideal solution for cloud computing. GreyCloud is a web based application created with Python and the web framework Django. It can be hosted on Linux with the Apache web server. The integrated development web server makes it easy to test applications and databases for GreyCloud on a local computer.

What can I do with GreyCloud?


GreyCloud offers a central platform for your services and databases. This means you can access to services through a SDK or native code with the HTTP protocol. It is even possible to use a web browser to send SQL queries to a database or call a web service.
