Represents a xml tree for service calls.

The xml tree has to have one of the following schemes:
<service_response>
    <error>
<xml_...>...</xml_...>
...
    </error>
    <result>
<xml_...>...</xml_...>
    </result>
</service_response>
or

<serialized_data>
    <xml_...>...</xml_...>
</serialized_data>
or

<service_request>
    <argument>
<xml_...>...</xml_...>
    </argument>
    <argument>
<xml_...>...</xml_...>
    </argument>
    ...
</service_request>
